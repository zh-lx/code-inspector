# 快速选择元素

当满足以下任一条件时，插件进入元素选择状态：
- 按住组合键（Mac 默认 `Option + Shift`，Windows 默认 `Alt + Shift`）
- 功能开关处于开启状态（彩色）

在该状态下，你可以用以下方式更快选中目标元素。

## 方式 1：鼠标移动选择

移动鼠标即可显示当前元素遮罩层。这是最直接的选择方式。

## 方式 2：滚轮切换父子元素

当当前元素已被高亮时，可通过滚轮切换层级：
- 向上滚动：切换到父元素
- 向下滚动：切换到子元素

![wheel](https://cdn.jsdelivr.net/gh/zh-lx/static-img/code-inspector/wheel.gif)

## 方式 3：DOM 树选择

按住组合键并在页面右键，可打开 DOM 树面板，在树中直接选择任意元素触发功能。

![dom-tree](https://cdn.jsdelivr.net/gh/zh-lx/static-img/code-inspector/dom-tree.gif)
